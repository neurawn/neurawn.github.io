**[GitHub](https://github.com/neurawn/neurawn.github.io)** &mdash; Jekyll &middot; GitHub Pages &middot; Markdown

Meta entry: the site you're reading right now.

Built on the [academicpages](https://github.com/academicpages/academicpages.github.io) Jekyll template (forked from [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/)) and hosted for free on GitHub Pages. I stripped out the parts that felt too academic-CV-ish and tried to make it feel more like a personal corner of the web.

If you want to build something similar, the academicpages template is a solid starting point — it handles most of the Jekyll boilerplate so you can focus on content.
